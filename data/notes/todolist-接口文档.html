1. 文档说明

  • 项目名称：ToDoList
  • 技术栈：Spring Boot 3 + MyBatis-Plus + Sa-Token + MySQL + Redis
  • 接口基础地址：http://localhost:8080
  • 文档格式：HTTP + JSON
  • 字符编码：UTF-8

2. 认证说明

项目使用 Sa-Token 做登录认证。

  • 登录接口：POST /user/login
  • 注册接口：POST /user/register
  • 除登录和注册外，其余接口默认都需要登录后访问
  • 请求头中的 Token 名称为：Authorization

请求头示例：

Authorization: 你的token
Content-Type: application/json

3. 通用返回格式

所有接口统一返回 Result<T> 结构：

{
  "code": 200,
  "message": "操作成功",
  "data": {}
}

字段说明：

字段名 类型 说明
code Integer 业务状态码
message String 提示信息
data Object 实际返回数据

常见状态码：

状态码 说明
200 请求成功
401 未登录，请先登录
500 业务异常或系统异常

4. 接口列表

模块 接口 方法 说明 是否需要登录
用户 /user/register POST 用户注册 否
用户 /user/login POST 用户登录 否
用户 /user/{name} GET 根据用户名模糊查询用户 是
用户 /user/{id} DELETE 删除用户 是，且需要管理员角色
待办 /todo/list GET 获取全部待办事项 是
待办 /todo/add POST 新增待办事项 是
